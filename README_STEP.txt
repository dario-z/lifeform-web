DIGITAL LIFEFORM — MOBILE DRAGGABLE SPLIT

Modifiche:
- Etichetta emotiva piccola nell'angolo superiore sinistro dello sprite.
- Ancora trascinabile tra sprite e chat.
- Split mobile regolabile dal 28% al 72%.
- Preferenza salvata nel localStorage del dispositivo.
- Doppio tocco/click sull'ancora: ritorno al 50/50.
- Accessibilità da tastiera:
  Freccia su/giù = modifica di 2 punti
  Home/End = minimo/massimo
  Invio/Spazio = ripristino 50/50

Installazione:
1. Estrarre nella root C:\Projects\lifeform-web
2. Confermare la sovrascrittura di:
   src\components\LifeformChat.tsx
   src\components\MobileChatLayout.css
3. npm run build
4. npm run dev
5. CTRL+F5 oppure nuova scheda privata sul telefono.

Non servono modifiche SQL.
