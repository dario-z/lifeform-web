DIGITAL LIFEFORM — MOBILE VARIABLE SPRITE + FULL-SCREEN CHAT

Nuovo comportamento:
- Lo sprite può occupare dal 24% al 96% dell'altezza dello schermo.
- La chat mantiene sempre un'altezza completa di 100dvh.
- Trascinare l'ancora modifica solo lo spazio dello sprite.
- Aumentando lo sprite, chat e pulsanti vengono spostati più in basso.
- La preferenza resta salvata nel browser.
- Doppio tocco/click sull'ancora ripristina lo sprite al 50%.
- L'etichetta emotiva resta piccola in alto a sinistra.

Installazione:
1. Estrarre nella root C:\Projects\lifeform-web
2. Sovrascrivere:
   src\components\LifeformChat.tsx
   src\components\MobileChatLayout.css
3. npm run build
4. npm run dev
5. CTRL+F5 oppure nuova scheda privata sul telefono.

Nessuna modifica SQL richiesta.
