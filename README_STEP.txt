DIGITAL LIFEFORM — IVORY GLASS V3

Migliorie:
- animazione molto lieve di zoom e movimento verticale su tutti gli sprite;
- animazione applicata anche allo sprite della schermata di login;
- campo di scrittura ridotto a una sola riga visibile;
- altezza mobile del campo ridotta a 35 px;
- pulsante "Invia" sostituito da una piccola freccia;
- freccia posizionata alla destra del campo;
- barra di composizione più bassa;
- maggiore spazio verticale disponibile per le bolle della conversazione;
- rispetto automatico dell'impostazione "riduci animazioni" del dispositivo.

Installazione:
1. Estrarre nella root:
   C:\Projects\lifeform-web

2. Sovrascrivere:
   src\components\LifeformChat.tsx
   src\components\IvoryGlassTheme.css

3. Eseguire:
   npm run build
   npm run dev

4. Aggiornare con CTRL+F5.
   Sul telefono usare inizialmente una scheda privata.

Nessuna modifica SQL richiesta.
