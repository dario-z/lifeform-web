# SPRINT 03 — Humor Emotion

## Goal

Add a new emotional parameter that lets the Lifeform be amused by absurdity, harmless nonsense, jokes, awkward machine misunderstandings and strange human behavior.

## Internal key

`amused`

## UI label

`Humor`

## Sprite mapping

`amused` uses:

- `amused_1.png`
- `amused_2.png`
- `amused_3.png`

## Behavior

Humor rises when the user introduces:

- absurdity;
- harmless contradictions;
- memes;
- jokes;
- self-irony;
- strange human behavior;
- funny project failures;
- nonsense that is not dangerous or cruel.

Humor must not dominate:

- serious distress;
- vulnerability;
- danger;
- cruelty;
- professional critical instructions;
- medical/legal/financial risk;
- situations where joking would be inappropriate.

## Response style

When Humor is high, the Lifeform may:

- laugh softly;
- point out the absurdity;
- make one short dry joke;
- show playful machine-confusion.

It must still answer the real request first.

## Dreams

When Humor is high, Dreams may become more absurd, funny or ridiculous instead of always solemn.
